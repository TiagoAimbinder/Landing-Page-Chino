# Django PageChino v1

## Como levantar pagina localmente

* Crear un python environment ( opcional )
  * virtualenv o virtualenvwrapper
* Instalar requirements.txt 
  * `pip install -r requirements.txt`
* Correr pagina web
  * `python manage.py runserver`
